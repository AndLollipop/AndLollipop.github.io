#### 斐波那契数列
斐波那契数列是递推算法的经典例子，典型的例子是兔子产仔的问题上：如果一对两个月大的兔子以后每一个月都可以生一对小兔子，而一对新生的兔子出生两个月后才可以生小兔子。那么假定在一年内没有发生兔子的死亡事件，那么一年后共有多少对兔子呢？
解答：
使用递推来解决，最终得出F(n) = F(n-2) + F(n-1)
代码：
```java
int Fibonacci(int n){ // n为月数
  int t1,t2;
  if(n == 1 || n == 2){ // 当n为1或者2的时候都为1对兔子
    return 1;
  }else{
    t1 = Fibonacci(n-1);
    t2 = Fibonacci(n-2);
    return t1 + t2;
  }
}
```
